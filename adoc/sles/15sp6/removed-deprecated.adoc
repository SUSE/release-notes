[#jsc-PED-8087]
=== intel-opencl and intel-graphics-compiler packages deprecation

Since SLE15 SP5, Intel stopped requesting updates intel-opencl and intel-graphics-compiler packages, the packages have been deprecated. They will be removed in {next-version}, and moved to Package Hub.

[#bsc-1230914]
=== NFSv2 is going to be dropped and unsupported

Currently, upstream has already started disabling NFSv2, so NFSv2 will be unsupported in SLE16. The below are the current status and plan for NFSv2

* SLE15: NFSv2 is enabled but depreciated.

* SLE16: NFSv2 will be disabled and dropped.
