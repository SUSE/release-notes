[#removed-deprecated-removed]
=== Removed Features and Packages

The following packages have been removed in this release.

* `libibmad-devel`
* `libipmctl2`
* `libqpdf13`
* `libreoffice-base-drivers-mysql`
* `libsmbldap0`
* `libsmbldap0-32bit`
* `openscap-extra-probes`
* `opensm-libs3`
* `opensm-libs3-32bit`
* `postgresql96`
* `postgresql96-contrib`
* `postgresql96-devel`
* `postgresql96-docs`
* `postgresql96-server`
* `vaapi-intel-driver`
* `xmlsec1-libgcrypt-devel`
