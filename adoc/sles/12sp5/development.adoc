[#jsc-SLE-12474]
==== PHP Has Been Updated to Version 7.4

We upgraded PHP to version 7.4 to provide you with the latest release.
To learn more about PHP version 7.4, we recommend reading the link:https://www.php.net/releases/7_4_0.php[PHP release announcement] and the link:https://www.php.net/manual/en/migration74.php[7.3.x to 7.4.x migration guide].

As of January 2021, PHP 7.2 is no longer supported.
For more information, see https://scc.suse.com/docs/lifecycle/sle/12/modules.

In {product} 12 SP5, we are enabling the latest Python 3.6 development which also enables machine-learning applications.
The Python 3.6 interpreter is shipped in the `python36-base` package.

For more information, see the upstream release notes at https://docs.python.org/3/whatsnew/3.6.html.
