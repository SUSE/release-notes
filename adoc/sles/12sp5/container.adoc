[#jsc-ECO-338]
==== New package: container-diff

The new package provides the command line tool `container-diff`. It allows
to analyze and compare certain criteria of container images including:

* Docker Image History
* Image file system
* Image size
* various software packages (RPM, apt, pip, npm)

These analyses can be performed on a single image, or a diff can be performed
on two images to compare. The tool helps to better understand what is
changing inside their images, and provides an overview of an image contains.
